# Memory Card Game

https://www.theodinproject.com/lessons/react-new-memory-card

Created React App using Vite.

## Requirements

The application should:
* Have a board with cards that are fetched from external API (capy.lol in this case)
* Have a score
* Implement rules of Memory Game

## Setup

1. Clone the project
2. `npm install`
3. `npm run dev`
